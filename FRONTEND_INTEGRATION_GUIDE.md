# Arohana Frontend-Backend Integration Guide\n\n## API Base URL\nhttps://api.arohanahealth.com\n\n## Key API Endpoints\n\n### Authentication\nPOST /auth/login - Submit {email, password}\nPOST /auth/register - Create new account\nGET /auth/profile - Get user profile\n\n### Bookings\nGET /bookings - List all bookings\nPOST /bookings - Create new booking\nGET /bookings/:id - Get booking details\n\n### Therapists  \nGET /therapists - Get all therapists\nGET /therapists/:id - Get therapist profile\nGET /therapists/:id/availability - Get availability\n\n### Sessions\nGET /sessions - List sessions\nPOST /sessions - Create session\nPUT /sessions/:id - Update session notes\n\n## Implementation Steps\n\n1. Set Backend URL in environment variables\n2. Create API client with fetch/axios\n3. Add JWT token handling for authentication\n4. Implement service functions for each feature\n5. Call API endpoints from frontend components\n\n## Hostinger Integration\n\nUse JavaScript fetch API to call the backend from Hostinger:  \n```\nconst therapists = await fetch('https://api.arohanahealth.com/therapists').then(r => r.json());\n```\n\n## Authentication\n\nStore JWT token in localStorage after login:\nlocalStorage.setItem('authToken', token);\n\nInclude in requests:\nAuthorization: Bearer <token>\n\n## Status\n✓ Backend deployed at Render (https://api.arohanahealth.com)\n✓ Database: PostgreSQL with Sequelize ORM\n✓ Features: Auth, Bookings, Sessions, Therapists, Email Notifications\n✓ Security: JWT, Rate Limiting, CORS, Input Validation\n\nFrontend can now integrate with these endpoints.
